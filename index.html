Developer Search Tool - OutSystems Platform

Team builder tool which let the user have a human resources database, with which the user can assign the most adequate employees to certain projects. The user can choose search parameters, such as showing only employees that know certain skill or have a rating superior to a value.

The web application was created through the OutSystems Platform, an IDE made by OutSystems. It's a simple, but powerful tool that lets almost everyone create an application with average knowledge of coding.

The subject was lectured by Dr. João Paulo Fernandes, and the Software Engineering project was proposed in partnership with OutSystems. The OutSystems' project supervisor was Vera Tiago. An OutSystems Platform professional training was given to the students attending the class in order to understand the IDE, with the possibility to take a certificate free of charge, which otherwise would be expensive to take. The courses of the training are free, online, and the platform is available for personal use.

  • Date: 2015
  • Fields: Web Design, Databases, Software Engineering
  • OutSystems: OutSystems
